🚌 Bus Booking System — Frontend
---
📌 Overview
This is the frontend of a Bus Booking System that allows users to browse routes, book tickets, and manage reservations through a simple and responsive interface.

The project focuses on user experience, clean UI design, and smooth navigation.

---
- 🚀 Features
- 🔍 View available bus routes
- 🪑 Select seats and book tickets
- 📅 Choose travel dates
- 📄 View booking details

---

## ⚙️ Installation & Setup
Clone the repository: git clone <https://github.com/camillebercilla/final-project--Tech_Vanguard--BSIT2B/tree/main/frontend>

git clone <your-repo-url>
Navigate to the frontend folder:

cd frontend
Open in browser:

index.html
🔗 Integration (Backend)
This frontend is designed to connect with a backend API.

Example:

fetch("http://localhost:5500/api/bookings")
Make sure your backend server is running.
